## About
I’m a Machine Learning Engineer focused on **shipping production AI** (LLMs, deep learning, RAG, and vision) with strong **MLOps / DevOps** instincts: cost-aware infrastructure, CI/CD, and reliable deployment.

**Recent focus:**
- Production AI microservices (vision + LLM reasoning) and full-stack delivery (API + app integration)
- Fine-tuning & evaluation workflows (LoRA/PEFT), CUDA optimization, and applied research
- Low-latency retrieval/search using relational + vector databases
